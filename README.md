##Features
- Tracks key presses and releases
- Creates a txt file that has a log of the keys pressed
- Adds timestamps to text for a detailed log
- Tracks special keys such as enter to create a new line, tab to create a tab space, spacebar to create a space, and backspace to delete
- Tracks 10 characters at a time before creating a new line in log
